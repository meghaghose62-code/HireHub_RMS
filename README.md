# 🏢 HireHub - Recruitment Management System

> A full-featured Recruitment Management System to streamline hiring, job posting, applications, interviews, and reports — all in one place.

---

## 📸 Preview

> Open the live link to see the full project in action!

---

## ✨ Features

- 👤 **User Management** — Admin, Recruiter, and Applicant roles
- 💼 **Job Posting** — Post, edit, and manage job listings
- 📝 **Applications** — Apply for jobs and track application status
- 🗓️ **Interview Scheduling** — Schedule and manage interviews
- 📊 **Reports & Analytics** — Job reports, user reports, and analysis
- 🔔 **Notifications** — System alerts and notifications
- ⭐ **Company Reviews** — Review and rate companies
- 💾 **Resume Upload** — Upload and manage resumes
- ✅ **Job Approval** — Admin approval system for job posts
- 🎯 **Recruiter Dashboard** — Dedicated recruiter panel
- 👨‍💼 **Admin Dashboard** — Full admin control panel

## 🚀 Getting Started

### Option 1 — Open directly in browser
Just open `index.html` in your browser — no setup needed!

### Option 2 — Use Live Server in VS Code
1. Install **Live Server** extension in VS Code
2. Right click on `index.html`
3. Click **"Open with Live Server"**
